# Overview

## Name
**Vamos** — the universal padel celebration shout.

## Vision
A fun, easy-to-use web app for keeping score in friendly Padel matches. The app features a beautiful courtside scoreboard display and a remote control that lets you update the score from any device — no need to walk to the phone.

## Goals
- Dead simple to use mid-game
- Gorgeous full-screen scoreboard for courtside display
- Remote control from a second device (phone, watch, tablet)
- Real-time sync between scoreboard and remote
- No app install required — just open a link

## Tech Stack
- **Framework:** Next.js (React)
- **Real-time sync:** Supabase Realtime
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **Language:** TypeScript
